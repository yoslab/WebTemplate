
## これは何？

- **Webの作成の手始めに！**テンプレートを置いています．
- Q. ダウンロードするには？
  - 右のDownload ZIPを選択し，ダウンロードしてください．
  - gitコマンドが使用できる場合は，以下を叩きましょう．

        git clone https://github.com/yoslab/WebTemplate.git
