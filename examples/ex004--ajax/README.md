
# **ajax**
JavaScriptを用いて，動的にPHPを実行する際のテンプレート集です．

----
## Ajaxを利用する
- JQueryを使う場合は，Ajaxを利用しましょう．
- [test_ajax.html](https://yoslab.github.io/WebTemplate/examples/ex004--ajax/test_ajax.html)のページで，[post/get_text.php](https://github.com/yoslab/WebTemplate/examples/ex004--ajax/post/getText.php)の出力内容を動的に取得します．

----
## Axiosを利用する
- JQueryを使わない場合は，よりモダンなAxiosを利用しましょう．
- [test_axios.html](https://yoslab.github.io/WebTemplate/examples/ex004--ajax/test_axios.html)のページで，[post/get_text.php](https://github.com/yoslab/WebTemplate/examples/ex004--ajax/post/getText.php)の出力内容を動的に取得します．
